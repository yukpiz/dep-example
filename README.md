## Resolve dependency.

```bash
$ go get github.com/yukpiz/dep-example
$ cd $GOPATH/src/github.com/yukpiz/dep-example/
$ dep ensure
```

## Quickstart `dep`

```bash
$ go get github.com/golang/dep
$ cd $GOPATH/src/github.com/{YOUR_REPOSITORY}/
$ dep init # Run in project root directory on $GOPATH.
```
